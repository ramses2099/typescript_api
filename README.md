# typescript_api

TypeScript Api

## Create the package file

npm init -y

## Install TypeScript

npm i -D typescript

## Create the tsconfig.json file

npx tsc --init

## Enable in tsconfig file sourceMap

"sourceMap": true,

## Install Express

npm i express
npm i -D @types/express

npm i cookie-parser
npm i -D @types/cookie-parser

## Compile app

npx tsc -w

## Run App

npm start
